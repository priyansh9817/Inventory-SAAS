# Inventory Management System

A full-stack Inventory Management System developed to manage products, stock, transactions, suppliers, and business operations efficiently.

---

# 🚀 Features

* Product Management
* Stock Management
* Inventory Tracking
* Transaction Management
* Dashboard Analytics
* User Authentication
* Role-Based Access Control
* Supplier Management
* Sales & Purchase Records
* Responsive UI Design

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Tailwind CSS
* Axios
* React Router

## Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

## Install Dependencies

### Backend

```bash
cd backend
npm install
```

### Frontend

```bash
cd frontend
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=
JWT_SECRET=
```

---

# ▶️ Run Project

## Backend

```bash
cd backend
npm run dev
```

## Frontend

```bash
cd frontend
npm start
```
